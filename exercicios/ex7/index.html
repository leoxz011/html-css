negrito e italic

negrito/destaque

para colocar em negrito usa se a tag [i] que nao é semantica ou a tag[strong] que é semantica

italic/destaque

para colocar em italic podemos usar a tag [i] que não é semantica ou a tag [em] que é semantica

marca texto

para deixar uma pate do texto marcado podemos usar a tag [mark]

texto pequeno

tambem podemos deixar o texto pequeno usando a tag [small]

texto riscado

para deixar um texto riscado usamos a tag [del]

texto sublinhado

para colocar o texto sublinhado utilize a tag [ins]

numero ao quadrado

só usar a tag [sup] x2+5

formulas

e para as formas a tag [sub] h2o

citaçoes

para fazer uma citaçao use a tag [q]

citaçao mais longa ou em blocos

para citaçoes em blocos a tag e a [blockquote]
